# 🚛 Tr.Alienigena — A Voz do Motorista

Aplicativo para conectar caminhoneiros a serviços, vagas, eventos e mais.

## Tecnologias
- HTML5, CSS3, JavaScript (modular)
- Firebase (Auth, Firestore)
- Leaflet (mapas)
- Cloudinary (upload de imagens)
- Vite (bundler) – em breve

## Estrutura
- `src/` – código fonte modular
- `public/` – assets estáticos

## Como rodar
1. Clone o repositório
2. `npm install` (em breve)
3. `npm run dev` (em breve)
